# HTML Basics

This section covers the fundamental concepts of HTML.

The projects in this directory introduce the core building blocks of web pages, including document structure, text formatting, lists, links, images, and tables.

## Projects

* Hello World
* Basic Document
* Headings
* Paragraphs
* Text Formatting
* Lists
* Nested Lists
* Hyperlinks
* Images
* Figure
* Tables
* Complex Tables
* Comments

## Learning Objectives

* Understand the structure of an HTML document.
* Learn common HTML elements.
* Create and organize web page content.
* Work with text, links, images, and tables.
* Build a solid foundation for CSS and JavaScript.

* HTML5
