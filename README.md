# Frontend Learning Journey

A structured collection of frontend development exercises and mini-projects built while learning modern web development from the ground up.

## Goals

* Learn HTML, CSS, and JavaScript step by step.
* Build a strong understanding of frontend fundamentals.
* Practice by creating real examples instead of only reading documentation.
* Track learning progress through hands-on projects.
